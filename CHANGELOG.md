# Changelog

Все заметные изменения LCBridge документируются здесь.
Формат основан на [Keep a Changelog](https://keepachangelog.com/),
проект придерживается [семантического версионирования](https://semver.org/lang/ru/).

## [1.1.0]
### Добавлено
- WebSocket-сервер для трансляции игровых данных в реальном времени.
- Колода монстров: присутствующие враги и самый частый монстр.
- Определение погоды с поддержкой WeatherTweaks / WeatherRegistry.
- Аналитика забега: деньги, время, смерти, разбивка по квотам.
- Маркеры турелей для отрисовки в оверлее.
- HTML-оверлей для OBS (Browser Source).

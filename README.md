# LCBridge

**Мост между Lethal Company и стрим-оверлеем в реальном времени.**

LCBridge — это плагин на BepInEx, который в реальном времени собирает игровые данные из Lethal Company и отдаёт их по WebSocket. На другом конце — HTML-оверлей, который рисует эти данные поверх стрима: колоду монстров, погоду, аналитику забега и прочее. Сделано для турниров и стримов сообщества **GDLP** (*Games Don't Like People*).

---

## Что входит в репозиторий

| Папка | Что внутри |
|-------|-----------|
| `BepInEx/plugins/` | Собранный плагин `LCBridge.dll` — кладётся в игру |
| `overlay/` | `overlay.html` — стрим-оверлей, который принимает данные мода |
| `thunderstore/` | Файлы для будущей публикации на Thunderstore (`manifest.json`, `icon.png`) |
| `src/` | Исходный код плагина (C#) |

---

## Возможности

- **WebSocket-сервер** внутри игры — отдаёт состояние игры оверлею в реальном времени.
- **Колода монстров** — какие враги присутствуют на луне, кого видели чаще всего.
- **Погода** — определение текущей погоды, поддержка WeatherTweaks.
- **Аналитика забега** — деньги, время, смерти, разбивка по квотам.
- **Дополнительные маркеры** — турели и прочие объекты для отрисовки в оверлее.

Плагин патчит `PlayerControllerB` и `StartOfRound`, читает состояние и транслирует его наружу, ничего не меняя в самой игре.

## Установка (для игроков)

> Требуется [BepInEx](https://thunderstore.io/c/lethal-company/p/BepInEx/BepInExPack/) (BepInExPack для Lethal Company).

**Вариант А — вручную:**
1. Установи BepInExPack для Lethal Company.
2. Скопируй `LCBridge.dll` в папку `…/Lethal Company/BepInEx/plugins/`.
3. Запусти игру один раз, чтобы сгенерировался конфиг.

**Вариант Б — через мод-менеджер** (r2modman / Thunderstore Mod Manager):
после публикации на Thunderstore — установить как обычный мод (зависимости подтянутся сами).

## Настройка оверлея (для стримера)

1. Открой `overlay/overlay.html` — это источник для OBS.
2. В OBS: **Источники → + → Браузер (Browser Source)**, укажи локальный файл `overlay.html`, размер `1920×1080`.
3. Запусти Lethal Company с установленным LCBridge — мод поднимет WebSocket-сервер, оверлей подключится к нему автоматически.
4. Положи источник оверлея **поверх** игрового захвата в сцене OBS.

### Конфигурация

После первого запуска игры BepInEx создаёт файл конфига в `BepInEx/config/`. Там можно поменять параметры соединения (порт WebSocket, интервал отправки данных и т.д.). Если меняешь порт — укажи тот же порт в настройках оверлея.

## Сборка из исходников

Проект на C#, таргет **.NET Standard 2.1**, зависит от BepInEx и сборок игры (`Assembly-CSharp`).

1. Открой проект в `src/` (Visual Studio / Rider / `dotnet`).
2. Пропиши пути к игровым сборкам и BepInEx (обычно через `Lethal Company.csproj` или `.csproj.user`).
3. Собери в Release — на выходе `LCBridge.dll`.

## Совместимость

- Lethal Company + BepInEx.
- Поддержка модов погоды: **WeatherTweaks / WeatherRegistry** (mrov).

## LCBridge v1.1.0 🛰️

Мост между **Lethal Company** и стрим-оверлеем: плагин в реальном времени собирает
данные забега и отдаёт их по WebSocket, а HTML-оверлей рисует их поверх стрима.
Все правки сделаны поверх read-only геттеров игры — **в игру ничего не пишется**.

### ✨ Что внутри

**Монстры**
- Единый резолвер имён врагов (`EnemyResolver`) — имена согласованы с иконками оверлея.
- Поддержка ToilHead: турель на голове добавляет суффикс (`Spring → Spring+Turret`, `Manticoil → Manticoil+Turret`). Без ToilHead работает как обычно.
- Подсчёт **реального числа уникальных особей** за забег (по instanceID); список сортируется по количеству.

**Ловушки**
- Определение ванильных ловушек — турели, мины, шипастые потолки — и передача их в оверлей (`"traps":[…]`).

**Лут (накопительный)**
- Сумма собранного скрапа считается по уникальным предметам и **не падает после продажи Компании** — деньги по квотам и профит по лунам считаются корректно.

**Погода и события**
- Определение текущей погоды с поддержкой **WeatherTweaks / WeatherRegistry**.
- Мягкая интеграция с событиями **Brutal Company**.

**Аналитика забега**
- Деньги, время, смерти (суммарно по всем игрокам), разбивка по квотам.

### 📦 Установка

1. Установи [BepInExPack для Lethal Company](https://thunderstore.io/c/lethal-company/p/BepInEx/BepInExPack/).
2. Скопируй `LCBridge.dll` в `…/Lethal Company/BepInEx/plugins/`.
3. Запусти игру один раз для генерации конфига.

### 🎥 Оверлей

Файл оверлея — в папке [`overlay/`](../../tree/main/overlay). Добавь `overlay.html`
в OBS как **Browser Source** (1920×1080), запусти игру с модом — оверлей подключится
к WebSocket автоматически.

### 🔗 Зависимости

- Мягкие (необязательные): BrutalCompanyMinusExtraReborn, WeatherTweaks
